
## API - Sistema de vendas com cadastro de produtos e registro de vendas

### Requisitos
*1.* Certifique-se de que tenha Java versão 21 instalada;  
*2.* Certifique-se de que tenha Docker instalado.

### Instruções para Execução:

- Abrir o Docker e manter a engine rodando;  
  

- Digitar o seguinte comando no terminal para montar a imagem do PostgreSQL no Docker:  
```docker compose up -d```  
  

- Subir a aplicação com Maven:  
```"mvn spring-boot:run"```  
  

- Abrir o arquivo Postman Collection, localizado em:  
```api-crud-venda-produto/RequisicoesAPI.postman_collection.json```   
   

- Realizar as requisições.
